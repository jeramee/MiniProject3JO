## Pip install instructions

### Please run the following:
```
pip install -r requirements.txt
```

### Install Flaskr:
```
    $ pip install -e .
```

### Run
```
    $ flask --app flaskr init-db
    $ flask --app flaskr run --debug
```

Open http://127.0.0.1:5000 in a browser.
